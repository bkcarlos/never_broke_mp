# 上线准备

## 小程序审核

1. 完善小程序基本信息：类目建议选择「工具 > 效率」或符合记账/工具属性的类目（以平台最新规则为准）。
2. 配置用户隐私保护指引，说明收集字段（openid、昵称等）与用途。
3. 提供测试账号说明（如需要）：云开发环境下测试号与操作路径。

## 文案与合规

- [docs/LEGAL.md](./LEGAL.md) 中用户协议与隐私政策需替换为正式法律文本。
- 薪资计算页已含「仅供参考」声明，发布前请法务复核。

## 订阅消息

1. 公众平台申请订阅消息模板（预算、分期、周期收入）。
2. 将模板 ID 填入 `miniprogram/config/subscribe.js`。

## 运维

- 定期导出云数据库备份。
- 关注云开发配额（调用次数、存储、带宽）。
