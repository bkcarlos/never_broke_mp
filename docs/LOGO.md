# NeverBroke Logo 与简介文案

## 一、Logo 设计说明

### 1.1 设计说明
- **概念** 账本/钱币的简洁几何形 + 向上弧线，表达「从糊涂花钱到心里有数、现金流往上走」。
- **色彩** 主色 #1AAD19（与文档里的微信绿一致），搭配深灰图形，小尺寸也清晰。
- **气质** 可信、克制、偏工具感，避免花哨渐变。
- **主形**：环形/「e」形开放曲线 + 指向右上方的箭头，表达「现金流向上、从糊涂到清晰」。
- **白色高光**：位于环形左上内侧的小块高光，增加层次与辨识度。
- **背景**：对外使用时以**透明**或浅色底为主；源图中的深灰/近黑仅作衬底，**不属于品牌色**。

### 1.2 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主绿 | `#29A639` | 与位图量化主色一致，接近 `#28A745`，明亮、偏产品/自然绿 |
| 高光白 | `#FFFFFF` | 高光区域填充 |
| 文档/导航绿（可选） | `#1AAD19` | 需求文档中的小程序导航主色，偏深；若需与导航条完全一致可替换主绿 |

### 1.3 矢量版（SVG）原则

- **仅保留绿色与白色**图形区域，深色背景在 SVG 中**不绘制**（等价透明）。
- 标准 SVG：使用 `<path>` 矢量数据，**不嵌入** PNG/Base64。
- 从位图重新生成时，可使用仓库脚本（需本机安装 ImageMagick、potrace）：

```bash
./scripts/build-neverbroke-logo-svg.sh
```

### 1.4 资源文件（仓库内）

| 文件 | 说明 |
|------|------|
| `miniprogram/images/neverbroke-logo.png` | 主 Logo 位图源文件 |
| `miniprogram/images/neverbroke-logo.svg` | 双色矢量（绿 + 白） |
| `miniprogram/images/neverbroke-logo-28.png` | 28×28 像素导出 |
| `miniprogram/images/neverbroke-logo-108.png` | 108×108 像素导出 |
| `miniprogram/images/neverbroke-logo-144.png` | 144×144 像素导出（小程序头像等） |

### 1.5 使用注意

- 小程序 **Tab / 小图标** 建议使用已导出的 **28×28** 或按设计稿倍数切图。
- 公众平台 **小程序头像** 等平台规范以微信官方尺寸为准（常见为 **144×144** 等），可在主 PNG 上另导一版。
- 缩放时若需「等比居中、透明留边」，应对源图做 **contain** 再落画布，避免强行拉伸变形。

## 二、产品简介文案

以下文案可用于登录页、关于页、应用商店简介、运营物料等，可按场裁剪长度。

### 2.1 一句话（副标题）

让每一笔账，都变成可执行的下一步。

### 2.2 短简介（小程序「关于」/ 登录页下方等）

NeverBroke（认知翻身账）是一款个人记账与财务规划小程序。在这里你可以快速记账、管理多账户与信用卡额度、设置月度预算与分期，并结合周期收入与现金流预测，看清未来几周的钱够不够花——少在表格和多个 App 之间来回切换。

### 2.3 更短版（约 50 字内）

记账、预算、分期、周期收入与现金流预测，一站式看清收支与未来余钱。

### 2.4 英文一行（如需）

NeverBroke — clarity for your money, from ledger to forecast.

### 2.5 名称对照

- **英文产品名**：NeverBroke  
- **中文副名**：认知翻身账
