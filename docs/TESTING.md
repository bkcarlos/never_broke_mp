# 测试清单

## 功能测试

1. **登录**：未登录进入登录页，登录后进入首页。
2. **账户**：新建储蓄卡/信用卡，列表展示与总资产。
3. **记账**：支出/收入保存后账本当日列表与余额变化；转账两账户余额。
4. **预算**：设置本月预算，首页预算进度变化。
5. **分期**：预览期数；记一笔勾选分期后分期列表出现。
6. **报表**：月度汇总、分类条、6 月趋势。
7. **时间线**：历史汇总与单日明细弹窗；未来周期收入列表。
8. **周期收入**：新建、实现收入后账本增加收入且下次日期推进。
9. **薪资**：选择城市计算税后，保存方案与删除。
10. **现金流**：切换 30/60/90 天，摘要与简图。
11. **导入**：选择 UTF-8 CSV，预览后全量导入。
12. **导出**：选择日期范围生成链接，复制/打开。
13. **设置**：隐藏金额后首页金额脱敏；安全线参与现金流风险提示（云函数侧）。

## 性能与安全

- 云函数冷启动：首调可能较慢，可合并低频函数或保持最小依赖。
- 导入大文件注意云函数超时与包体，必要时改分片上传至云存储再解析。
- 删除交易当前不回滚余额，生产环境应实现差额调整或禁止删除仅允许冲正。

## 自动化

可在仓库根目录使用 `node scripts/check-structure.js`（若存在）校验页面路径与 `app.json` 一致。
