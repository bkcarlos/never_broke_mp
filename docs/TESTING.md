# 功能测试清单（NeverBroke）

上线前建议在真机与开发者工具各跑一遍。

## 认证与引导

- [ ] 未配置 `CLOUD_ENV_ID` 时登录有明确提示
- [ ] 勾选协议后可登录；拒绝头像昵称仍可登录
- [ ] 新用户弹窗「去创建账户」与「稍后」分支正常
- [ ] 云函数返回 401 时自动清本地用户并回登录页

## 首页 / 账本

- [ ] 下拉刷新首页数据
- [ ] 无账户时提示条与跳转新建账户
- [ ] 未设预算时展示「去设置」
- [ ] 记一笔：常用标签、编辑账单、删除与确认
- [ ] 账本：日期、类型/账户/分类筛选；转账仅删除无编辑

## 账户 / 预算 / 分期

- [ ] 新建/编辑账户；信用卡额度；归档
- [ ] 预算保存、历史列表、从上月复制、80%/超支弹窗（同月仅一次）
- [ ] 分期预览、列表、明细、标记已还

## 报表 / 时间线 / 现金流

- [ ] 报表月份切换；Canvas 饼图与折线图渲染
- [ ] 时间线历史 30/60/90；未来合并周期收入与分期
- [ ] 现金流 30/60/90；安全线在设置中配置后风险提示

## 周期收入 / 薪资 / 导入导出

- [ ] 周期收入到期标识、实现入账、删除
- [ ] 薪资计算与方案保存/删除
- [ ] CSV 导入预览与执行；导出快捷范围与下载/打开

## 设置

- [ ] 语言切换后 Tab 文案变化；隐藏金额全局生效
