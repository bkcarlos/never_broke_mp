## 阶段二：订阅消息配置（平台侧）

> 在微信公众平台后台完成。

### 2.1 创建订阅消息模板
- [ ] 登录微信公众平台 → 「订阅消息」
- [ ] 创建「预算提醒」模板，记录模板 ID
- [ ] 创建「分期还款提醒」模板，记录模板 ID
- [ ] 创建「周期收入提醒」模板，记录模板 ID

### 2.2 填写模板 ID
- [ ] 将 3 个模板 ID 填入 `miniprogram/config/subscribe.js` 的 `TEMPLATE_IDS` 数组
- [ ] 将 3 个模板 ID 填入 `miniprogram/config/subscribe.js` 的 `SERVER_TEMPLATE_IDS` 对象
- [ ] 将 3 个模板 ID 填入 `cloudfunctions/notify/subscribe.js` 的 `SERVER_TEMPLATE_IDS` 对象
- [ ] 重新部署 `notify` 云函数使配置生效

### 2.3 配置定时触发器
- [ ] 在云开发控制台为 `notify` 云函数创建定时触发器
- [ ] 建议cron：每天早上 9 点执行 `sendAll`
- [ ] 确认触发器已启用

### 2.4 验证订阅消息
- [ ] 在真机上进入设置页点击「请求订阅消息」
